Описание программы

Программа собирает данные о подарочных картах, доступных на сервис для мгновенной он-лайн отправки
подарочных карт Giftery.

 По каждой карте с сайта получаются следующие данные:
 1. Наименование карты
 2. описание карты
 3. доступный номинал карты
 4. ссылка на карту на сайте Giftery
